Archittetura Client-Server

Introduzione

L'architteura client server si riferisce quando in una rete un computer client si connette ad un server per usufruire di un servizio.

Client

Con il client, ci riferiamo al dispositivo Hardware o Software che manda una o più richieste a un server per usufruire di certi servizi. Quando usi gmail il tuo browser è il client mentre il server sono i sever di google che ti danno i dati su tutte le e-mail che hai ricevuto. Le tue email inviate e ricevute sono nel client localmente, tutto il resto delle email delle altre persone sono sul server.

Server

il server invece è un dispositivo che si occupa di gestire gli accessi dei client,allocare,rilasciare le risorse.un server web è anche associato a un DB dove vengono salvati i dati riguardo al servizio, come i dati d'accesso di ogni account,i dati degli account,reperti medici ecc. Ogni server ha una porta e un indirizzo ip che il client poi userà per stabilire una connesione. per approfondimenti puoi recarti qui.
